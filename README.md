# 去看看
* 一款在线小说阅读器，数据来源全部从去看看小说网抓取，详细功能如下：
  * 支持在线书籍查找
  * 网络书籍分类列表，排行列表
  * 在线阅读
  * 章节下载后离线查看
  * 阅读界面字体大小、颜色修改、背景颜色修改
  * 阅读历史自动保存
* 界面预览图
